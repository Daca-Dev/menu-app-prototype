# Menu app result [Prototype]

## Requirements

- Node v18.12.1 (nvm recommended)
- Yarn v1.22.19

## Setup

1. Validate if you have the correct Node versio, can use `nvm use` command to setup the correct one
2. execute `yarn install` command

## Development
